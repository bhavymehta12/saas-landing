# Leomora — AI Video Generator Landing Page

A stunning, aesthetic SaaS landing page built with pure HTML, CSS & JavaScript.

🔗 **Live Demo:** https://leomora.netlify.app

## Features
- Animated hero section with glowing orbs and grid background
- Scrolling marquee ticker
- Interactive feature cards with typing animation
- 3-step process section
- Pricing with monthly/yearly toggle
- Email waitlist form with success animation
- Scroll reveal animations
- Fully responsive (mobile friendly)

## Built With
HTML5 · CSS3 · Vanilla JavaScript · Google Fonts (Cormorant Garamond)
